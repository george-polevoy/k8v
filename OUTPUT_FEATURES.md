# Output Features

## Overview

k8v now includes comprehensive output visualization for each node. When you run a computation, you can see both text output (stdout/stderr) and graphics output for each node.

## Features

### Text Output
- Captures all `print()` and `log()` statements from node execution
- Displays errors and exceptions
- Shows in a collapsible panel with monospace font
- Dark theme for better readability

### Graphics Output
- Displays images generated by nodes
- Supports base64 encoded images
- Supports data URLs (data:image/...)
- Automatically scales to fit the panel
- Collapsible panel for space management

## Usage

1. **Select a node** on the canvas
2. **Run the graph** using the â–¶ button in the toolbar
3. **View outputs** in the Output Panel on the right side:
   - Text Output: Shows all printed text and errors
   - Graphics Output: Shows any generated images

## Code Examples

### Text Output Example
```javascript
// In a node's inline code:
print("Hello, world!");
print("The result is:", inputs.value);
outputs.result = inputs.value * 2;
```

### Graphics Output Example
```javascript
// Generate a simple graphics output (base64 encoded)
const canvas = document.createElement('canvas');
canvas.width = 200;
canvas.height = 200;
const ctx = canvas.getContext('2d');
ctx.fillStyle = 'blue';
ctx.fillRect(0, 0, 200, 200);
const dataUrl = canvas.toDataURL();
outputGraphics(dataUrl); // or outputImage(dataUrl)
```

## UI Layout

The Output Panel is positioned on the right side of the screen, next to the Node Panel:
- **Left**: Canvas with nodes
- **Middle**: Node Panel (for editing node properties)
- **Right**: Output Panel (for viewing results)

Both text and graphics sections are:
- **Collapsible**: Click the header to expand/collapse
- **Stacked**: Text output on top, graphics below
- **Auto-refreshing**: Updates when you run computations
